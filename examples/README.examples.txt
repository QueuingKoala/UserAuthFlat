UserAuthFlat examples README

Best viewed with word wrap enabled

OVERVIEW

These UAF examples are designed to demonstrate simple API caller usage.

EXAMPLE DESCRIPTIONS:

UAFSample_Add_User.au3				Add a user (basic version)
UAFSample_Add_User_Advanced.au3		Add a user (advanced version)
UAFSample_Verify_User.au3			Verify (authenticate) a user
UAFSample_Remove_User.au3			Remove a user
UAFSample_List_Users.au3			List users in Auth file
UAFSample_List_Formats.au3			List supported formats, hashes, & default rounds
